# Python 自主學習成果

## 學習內容

- Python 基礎語法
- Dictionary（字典）
- Set（集合）
- Tuple（元組）
- 類別與物件導向程式設計

## 作品成果

### 銀行帳戶管理系統
利用類別、繼承與方法覆寫的概念，模擬銀行存款與提款功能。

### 成績計算系統
利用 Dictionary 儲存學生成績資料，計算總分與平均成績。

### 大樂透號碼排序
利用 Set 及排序功能完成資料整理。

### 學期總成績計算
利用 Tuple 及權重計算各科加權後的學期成績。

## 學習收穫

透過 Python 自主學習，我逐步建立程式邏輯思維與問題分析能力，並學習資料結構及物件導向程式設計概念。未來希望能將程式設計能力與電子工程專業結合，發展軟硬體整合能力。
